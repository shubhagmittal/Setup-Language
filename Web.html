C-setup

Select your operating system:
  • MacOS
  • Windows
  • Linux

To setup C- language on MacOS you must follow these steps:

  • Firstly go to VS Code and download desired VS Code
  • After downloading install the application in you laptop
  • Once the application is installed go to extensions and install c/c++ extension pack
  • You are ready to code in C-Language

To setup C-Language in windows machine follow these steps :

  • Firstly go to VS Code and download desired VS Code
  • Now after installing VS Code go to extensions and install c/c++ extension pack
  • After installing extension pack download MinGW
  • Install all basic setup logs in MinGW
  • Now open VS Code and install a extension named code runner
  • Then go to File on top corner and click on prefrences
  • In the options select Settings
  • Now in the left side click on Extensions
  • From the dropdown search for Run in terminal and check the checkbox
  • Make a new C-Language file and write a code to run the code right click anywhere and select the first option
  • You are ready to code in C-Language

To setup C-Language in Linux follow the steps :

  • Firstly go to VS Code and download desired VS Code
  • After installing VS Code go to extensions and install c/c++ Extensions pack
  • Now go to terminal and run command :
    • sudo apt-get install build-essential -y
  • After completion of this installation go to VS Code and write a code in C-Language
  • Now open terminal in VS Code and run the command :
    • gcc "file-name" -oc
    • ./c
  • Now you are all set to code in C-Language

Python Setup

Select your operating system:
  • MacOS
  • Windows
  • Linux

Follow the steps to setup pyhton in MacOS :

  • Firstly go to VS Code and download desired VS Code
  • Now go to Python and download apropriate version
  • go to VS Code and install Python extensions pack
  • you are now ready to code in Python

To setup Python in ypur windows system :

  • Go to VS Code and download desired VS Code
  • Open microsoft store and search python
  • Click on get and install Python in your system
  • Open VS Code and install python extensions pack
  • you are now good to go with pyhton

To install python in your system follow the given steps :

  • Go to VS Code and download desired VS Code
  • Open your terminal and enter the code:
    • python3 --version
  • If it says any error then run :
    • sudo apt-get install pyhton3
  • Open VS Code and install python extension pack
  • your system is ready to code in python
